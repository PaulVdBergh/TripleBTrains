void TBT::Manager::setPowerState ( PowerState  newState)

sets the new powerstate and notifies all interfaces about the new state

Manager::setPowerState set the powerstate of the system to the new state indicated by newState and informs each interface about the new state. In the case that newState is PowerOn, this function will cancel any pending emergencystop.
